Organisation types represent the different types of businesses, departments, or other business units that your users belong to. This can be used to limit which organisations different types of users can sign up to.

For example, if you have a user group "pharmacist" you might only want them to register to organisations of type "pharmacy".

You can set the organisation type on your business units either one at a time through the Manage Business Units page, or you can set them when you upload a spread sheet of organisations.
