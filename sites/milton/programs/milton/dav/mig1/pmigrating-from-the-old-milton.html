Migrating from the old milton

In this tutorial we'll take the old milton reference implementation and show how to upgrade it to the latest milton 2.2.1.

Migration has these steps

  • Include the updated dependencies (jars)
  • Fix imports in your Resource implementations
  • Integrate the new servlet filter
