WEBDAV is like FTP but implemented in HTTP. It's a mature standard and supported natively in all operating systems.

You can go through the following links to know more about WebDAV.

  • http://en.wikipedia.org/wiki/WebDAV
  • http://www.webdav.org/

What is Milton

Milton is an API and HTTP protocol handler for adding WebDAV support to web applications. Milton is an open source java library for implementing desktop integration in your web apps with WebDAV.

Once you've integrated Milton into your application, users can view and/or manipulate remote data in your application using drag and drop of files and folders in their native OS file browser - eg Windows Explorer, Nautilus, etc, over HTTP.

Milton is normally used with a servlet, but it is decoupled from the Servlet API so it can easily be integrated into alternative web servers.
